# Reto Practicas

Proyecto de gestión de tickets: Frontend en Angular y Backend en Node/Express + PostgreSQL.


## Estructura
- `reto-practicas-frontend`
- `reto-practicas-backend`



## Cómo ejecutar

### Backend
cd reto-practicas-backend
npm install
node server.js

### Frontend
cd reto-practicas-frontend
npm install
ng serve
